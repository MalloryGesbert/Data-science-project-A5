## Installation

Créer votre environement virtuel :  

```bash
python -m venv env
```

Activer le :  

```bash
.\env\Scripts\activate
```

Installer les dépendances :  

```bash
pip install -r requirements.txt
```

## Section dev

Penser à update les dépendances avec :  

```bash
pip freeze > requirements.txt
```
